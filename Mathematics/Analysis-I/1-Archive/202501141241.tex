\section*{Main Content}
\textbf{Main Idea}\\
An ordered set is a set $S$ in which an order is defined.\\

\textbf{Explanation}\\
For example, $Q$ is an ordered set if $r < s$ is defined to mean that $s-r$ is a positive rational number.\\
